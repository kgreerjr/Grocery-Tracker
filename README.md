# Grocery-Tracker

A C++ application that analyzes grocery purchase data from a text file, tracking item frequencies and presenting results through searchable queries, formatted lists, and ASCII histograms.

## About

- Loaded grocery item data from an input text file into a std::map, counting purchase frequencies per item and writing the results to a backup data file for persistence across sessions.
- Implemented a menu-driven interface with three output modes: single-item frequency lookup, a formatted list of all items with counts, and an ASCII histogram displaying frequency bars.
- Separated program logic across a header file, implementation file, and main driver, demonstrating modular class design and clean separation of interface from implementation.
